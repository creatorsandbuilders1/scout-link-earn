# REFERYDO! Landing Page

## Overview
An immersive, scroll-driven narrative landing page that breaks away from traditional layouts to create an unforgettable first impression.

## Design Philosophy
**"The Vibrant Workspace"** - Bold, energetic, and unapologetically Web3. This landing page is an experience, not just a website.

- **Giant typography** with REFERYDO! branding

- **Electric blue gradient background** with dynamic elements
- **Fixed "Connect Wallet" button** always accessible in top-right

## Technical Features

### Animations
- **Scroll-triggered reveals** for progressive storytelling
- **Cursor-following effects** on hero section
- **Floating elements** throughout
- **Glassmorphism** with backdrop blur
- **Smooth transitions** between sections

### Color Palette
- **Electric Blue (#2563EB)** - Trust & Structure
- **Vibrant Orange (#F97316)** - Opportunity & Decision
- **Kinetic Green (#4ADE80)** - Growth & Success
- **Crisp White (#FFFFFF)** - Canvas
- **Deep Black (#000000)** - Contrast & Drama

### Responsive Design
- Mobile-first approach
- Breakpoints for tablet and desktop
- Horizontal scroll works on all devices
- Touch-friendly interactions

### Performance
- Optimized animations using CSS transforms
- Lazy-loaded effects
- Efficient scroll listeners
- No heavy libraries

## User Journey
1. **Impact** - Immediate visual wow factor
2. **Problem** - Relate to user pain points
3. **Solution** - Show the three-role ecosystem
4. **Trust** - Build confidence with guarantees
5. **Action** - Clear, compelling CTA

## Development Notes
- Uses existing REFERYDO design system
- Integrates seamlessly with dashboard routing
- Avatar images from DiceBear API (placeholder)
- All animations in pure CSS for performance
- Conditional navigation (landing vs. app)
